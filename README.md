# ⚡ NeoPulse — Y2K Music Player

A nostalgic, immersive music player inspired by Winamp and Windows Media Player from the 2000s era. Built with React + Web Audio API.

🔗 **Live Demo:** https://neopulse-player.vercel.app/

---

## ✨ Features

- 🎵 Local music playback (MP3, WAV, and more)
- 🎨 3 skins — Cyber, Luxe, Chrome
- 📊 7 visualizer modes:
  - Bars, Wave, Orbit, Mirror, Psychedelic, Inferno, Starfield
- ✨ Particle effects that react to the beat
- 🔊 Volume control + seek bar
- 📋 Playlist management
- 📺 Scanline + grid overlay for authentic Y2K feel

---

## 🚀 Run Locally

```bash
git clone https://github.com/Katlego2LO/neopulse-player.git
cd neopulse-player
npm install
npm run dev
```

Then open http://localhost:5173
